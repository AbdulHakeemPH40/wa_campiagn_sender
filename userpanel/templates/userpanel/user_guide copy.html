Step 2: Create WhatsApp Session & Scan QR

A session connects your WhatsApp account to WA Campaign Sender via QR code.

What is a Session?

Each session = one WhatsApp number connected. You can create unlimited sessions for different WhatsApp accounts.

How to Create & Connect:

  1. Go to WhatsApp Sessions Sessions Page
  2. Click "Create New Session" - Enter session name and optional phone number
  3. System generates QR code - Shows on connect page
  4. On your mobile:
    • Open WhatsApp app
    • Go to Settings → Linked Devices
    • Tap "Link a Device"
    • Point camera at QR code
    • Wait for status confirmation
  5. Status updates: pending → connecting → connected
  6. Ready to send campaigns! Session now active

Important Notes

  • Your WhatsApp number must match the one you scan with
  • Phone must stay online during scanning
  • Don't use WhatsApp Web on another browser simultaneously
  • Session remains active as long as connected

Step 3: Upload & Manage Contacts

Contacts are the people who will receive your WhatsApp messages.

Upload CSV or Excel File:

  1. Go to Contacts Page
  2. Click "Upload Contacts" button
  3. Select your CSV or Excel file
  4. System validates and imports automatically

CSV Format Example:

phone_number,first_name,last_name,company,country
917712345678,John,Doe,ACME Corp,India
447911123456,Jane,Smith,Tech Ltd,UK
12155552671,Bob,Johnson,USA Inc,USA

Required & Optional Columns:

Required
  • phone_number - With country code (e.g., +91)
Optional (for personalization)
  • first_name, last_name, email
  • company, city, country
  • Any custom fields you add

Step 4: Create & Send Campaign

Create Campaign:

  1. Go to Send Campaign Page
  2. Select Session: Choose a connected WhatsApp session
  3. Choose Contact List: Select from uploaded lists
  4. Campaign Name: e.g., "Holiday_Promotion_2025" (use underscores)
  5. Message: Write your message with variables
    Example: "Hi {first_name}, welcome to {company}!"
  6. AI Draft (Optional): Generate professional message with ChatGPT
  7. Attachment (Optional): Add image, video, or PDF
  8. Advanced Controls (Recommended): Configure batch settings

Message Variables:

Use these placeholders to personalize messages:

Important Safety Settings

Advanced Sending Controls (Highly Recommended)

Prevents WhatsApp account ban by simulating human-like sending patterns:

  • Message Delay: Random wait between messages (e.g., 5-30 seconds)
  • Batch Size: Send in groups (e.g., 50-70 contacts per batch)
  • Batch Cooldown: Wait between batches (e.g., 5-10 minutes)

Example: 500 contacts take ~110 minutes (1h 50m) with safety delays

AI Moderation Check

System automatically scans message for violations:

ALLOWED

Safe to send - no issues detected

REVIEW

Admin reviews before sending

BLOCKED

Illegal content detected - cannot send

Send Campaign

  1. Review all settings - Message, contacts, session, safety options
  2. Optional: Send Test Message - Try with one number first
  3. Click "Send Campaign" button when ready
  4. Campaign queues for background processing
  5. System starts sending with configured delays
  6. Monitor progress at Campaigns Page

Monitor Campaign

After sending, track real-time progress:

  • Live progress percentage and counters
  • Messages sent, delivered, read, failed counts
  • Current batch and next batch countdown
  • Pause or stop campaign anytime
  • Retry failed messages after completion
  • Export results to Excel for analysis

Account Warm-up Strategy

Account Age: 3+ months old (Recommended)

Never use brand new WhatsApp accounts for bulk sending.

Week 1: 50 messages/day

Start conservatively to establish account legitimacy

Week 2: 100 messages/day

Gradual increase - monitor for issues

Week 3: 200 messages/day

Continue gradual increase

Week 4+: 250-500 messages/day

After successful warm-up

Maximum Safe: 2,000-3,000 messages/day

DO NOT exceed 3,000/day - High risk of account suspension
